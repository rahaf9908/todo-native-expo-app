# Todo Native App (Expo)

React Native (Expo) conversion of the provided React web todo app.

## Features
- Add todo
- Update todo in a separate screen
- Delete todo in a separate screen (with confirmation)
- React Navigation for screen navigation
- Persistent storage using AsyncStorage

## Tech Stack
- Expo + React Native
- React Navigation (Native Stack)
- AsyncStorage

## Run Locally
```bash
npm install
npx expo start
